# ギフトの受け取り方法

<img width="777" height="778" alt="image" src="https://github.com/user-attachments/assets/1244b31e-2e8f-4785-819f-329aaeb8e3b2" />


リンクを押すと、表示されるページで「ギフトを受け取る」をクリックします。

すでに所有している商品を受け取らないように注意してください

また、古いギフトは削除される場合がありますので、できるだけ早めにご利用ください。


<img width="459" height="316" alt="image" src="https://github.com/user-attachments/assets/6e663d93-0b89-413e-bda4-5abeb3a7f3d4" />

ライブラリでギフト項目をクリックすると、受け取ったギフトを確認できます。
